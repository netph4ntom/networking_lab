## Overview

Repository ini berisi kumpulan praktik dan dokumentasi pembelajaran networking yang berfokus pada pemahaman konsep secara mendalam serta implementasi teknis melalui simulasi. Setiap lab dirancang untuk memahami cara kerja protokol jaringan, menganalisis output konfigurasi, dan melatih troubleshooting secara sistematis.

Topik yang dibahas mencakup fundamental hingga intermediate networking seperti switching, routing, VLAN, dan Spanning Tree Protocol. Seluruh praktik dilakukan menggunakan Cisco Packet Tracer, GNS3, dan lingkungan CLI untuk membangun kebiasaan kerja yang relevan dengan dunia industri.

Repository ini dibuat sebagai sarana pengembangan kemampuan teknis di bidang Network dan Infrastructure Engineering sekaligus sebagai dokumentasi progres belajar yang terstruktur dan profesional.
